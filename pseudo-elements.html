Mike McCoy: Pseudo Elements

Article 1: Understanding Pseudo-Element :Before And :After

hongkiat.com pseudo elements

This article explains the basics of pseudo elements and how they can let you add extra markup in CSS to an HTML markup document, instead of just using CSS for styling. It focuses on just the before and after elements to keep it simple for the reader. It explains the difference between pseudo elements and pseudo classes, denoted with the double colon. It sounds like it’s a fake element that behaves like a real element, that you can then style.


Article 2: CSS Pseudo Elements

cssbasics.com pseudo elements

This article covers similar ground. It explains the syntax difference compared to regular CSS in regards to the colon, similar to the first article. They do a nice job of categorizing everything, from the elements you can expect to deal with, to the actual lines of code you’ll need to plug in. They cover the use of the paragraph tag, which the previous article didn’t really get into. They also list out some of the types of elements you can use a pseudo element for. I’m still not totally clear on pseudo elements, but this was a good first exposure to the concept.
