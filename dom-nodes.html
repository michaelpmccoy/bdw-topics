Article 1: JavaScript and the DOM Series: Lesson 1/h2>

net.tutsplus.com DOM nodes

This article talks about JavaScript and the DOM API. It might be a bit beyond a beginner level, but they have a nice breakdown of what a DOM is in one section. They define it as the Document Object Model, which is the API that JavaScript uses to interact with a website’s content. This lets it access and affect the HTML. They include a graphic that shows a typical DOM outline. They also break down some of the typical types of nodes, like element, text and document.
